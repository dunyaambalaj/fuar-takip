🔥 Firebase Kurulum

Firebase bilgileri koda gömülüdür — farklı cihazlarda aynı veritabanı otomatik senkronize olur.

✅ Aktif proje: fuar-takip-e42a1 — Herhangi bir cihazdan açın, veriler anlık senkronize olur.
1
Firestore Kuralları (Firebase Console → Kurallar):
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }
2
Excel İçe Aktarma: "📥 Excel" → .xlsx veya .csv seçin.
Sütunlar: musteriAdi | firmaAdi | temsilci | il | ilce | tutar | odemeDurumu | tarih | notlar
3
Müşteri Bazlı Tahsilat: Müşteri satırına veya kartına tıklayın → Firma firma detay + Tümünü Tahsil Et butonu çıkar.
4
Mobil: Kart görünümü otomatik devreye girer. Kartlara tıklayarak müşteri detayına erişin.
⚠ Herkese açık Firestore kuralları kullanıyor — URL'yi sadece güvendiğiniz kişilerle paylaşın.
